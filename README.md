# my2048
Wechat small program to achieve 2048

## 项目介绍
是一个菜鸟边学边写的一个微信小程序项目

以一个2048小游戏作为微信小程序学习的开始

## 页面设置
page包括: 
1. 首页
2. 游戏主页面
3. 登录日志页面

其中设置首页和日志页面为tabbar
